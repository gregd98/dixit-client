# Dixit (client)
This is the **frontend** of the Dixit board game, LAN multiplayer version.

Server: [dixit-server](https://github.com/gregd98/dixit-server)

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Available Scripts
Install: `npm install`

Run: `npm start`

Build: `npm run build`

_To switch localhost to build version, you need to edit **src/constants.js**_
